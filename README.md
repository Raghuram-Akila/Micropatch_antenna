# Microstrip Patch Antenna Designs

This repository contains microstrip patch antenna designs developed using CST Studio Suite. The work focuses on designing, analyzing, and optimizing antennas for various RF and microwave applications.

## 📡 Features

* Single-band and multi-resonant antenna designs
* FR4 substrate-based implementations (εr ≈ 4.3)
* Coaxial probe-fed configurations
* Techniques for resonance tuning and unwanted frequency suppression
* Parametric analysis of antenna dimensions
* Exploration of advanced concepts such as:

  * Slot-based designs (π-slot, U-slot, etc.)
  * Miniaturization techniques
  * Gain and efficiency improvement methods

## 🛠 Tools Used

* CST Studio Suite
* (Optional) Comsol Multiphysics / Ansys HFSS for comparison

## 📂 Contents

* Antenna design files (.cst)
* Geometry and structure images
* S11 (return loss) plots
* Radiation patterns
* Design parameters and calculations

## 🎯 Objectives

* Achieve desired resonance frequencies
* Eliminate unwanted resonances
* Improve gain and radiation efficiency
* Develop compact and practical antenna structures

## 📖 Applications

* Wireless communication systems
* IoT devices
* RF and microwave research
* Academic and simulation studies
